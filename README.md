# GRC Engineering Capstone Portfolio

This repository contains machine-readable compliance evidence and Infrastructure-as-Code (IaC) patterns for a multi-cloud environment. 

## Included Labs
* **Lab 2.3 (AWS):** Compliant S3 Bucket (SC-28, AU-3, AU-6, CM-6, AC-3).
* **Lab 2.4 (GCP):** Compliant GCS Module with KMS & Attestation (SC-12, SC-13, SC-28, AU-11, CM-6).
* **Lab 2.5 (AWS):** IaC as Compliance Evidence 
## Project Structure
* `/modules`: Reusable compliant infrastructure blueprints.
* `/primitives`: Single-resource deployments.
* `/evidence`: JSON-formatted system-generated evidence (SGE) for audit verification.
